# Delta AI

مشروع ويب تعليمي ذكي يضم صفحات رئيسية للتسجيل، تسجيل الدخول، لوحة التحكم، المواد، التحديات والملف الشخصي.

## الهيكل
- HTML: الصفحات الرئيسية
- CSS: الأنماط العامة والصفحات الخاصة
- JavaScript: منطق المصادقة والتحديات ولوحة التحكم

## التشغيل
1. افتح ملف index.html في المتصفح.
2. أو استخدم أي خادم محلي بسيط مثل Live Server.

## ملاحظات
- يتم استخدام نموذج تجريبي للمصادقة حتى يتم ربط Firebase أو backend حقيقي.
